◆ getWaitingListCount()

void com.ijaskz.lotteryeventapp.WaitingListManager.getWaitingListCount ( String eventId,
OnCountListener listener )

Asynchronously counts how many entrants are on the waiting list for a given event.

Implementation detail: this executes a Firestore aggregation (count) on the waiting_list collection filtered by event_id. The result is delivered on the supplied OnCountListener callback.

Contract

  • If eventId is null or blank, the listener receives onCount(0).
  • On success, exactly one of onCount(int) is called.
  • On failure (e.g., networking / permission), exactly one of onError(Exception) is called.
Parameters
eventIdFirestore event document id to count entrants for
listenercallback invoked with either the count or an error
